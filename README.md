# Simple to do list PHP

## setup

  $sudo systemctl start mysql
  $sudo systemctl start apache2

  $sudo git clone git@github.com:nullx5/simple_todolist_php.git 
  $cd simple_todolist_php
  $sudo mysql -u root < /path/todolist.sql
